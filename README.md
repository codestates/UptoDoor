# 🏡 About
### 우리 동네에도 월 구독서비스가 있다는걸 아셨나요?
의류, 세탁, 반찬 등 <br/>
매번 챙기기 힘들지만 내 삶에 꼭 필요한 것들.<br/> 
우리동네 구독 서비스 중개플랫폼 **UptoDoor** 를 이용하여 번거로움을 줄이세요.

**❗️ 단순 구독서비스가 아니에요.**<br/>
소비자들은 내가 등록한 지역 반경 2.5km 내 가게의 구독서비스를 이용할 수 있어요.<br/>
주변에 좋은 월구독서비스가 존재하나 알지 못해 이용하지 못했던 소비자들에게는 정보의 플랫폼이 됩니다.<br/>
더불어 가까운 곳에서, 더 빠르고 신뢰도 있는 서비스를 받을 수 있어요.<br/>

**❗️ 지역경제를 생각해요**<br/>
코로나19로 온라인 구독서비스는 잘되는 반면, 지역경제는 침체되어 소상공인들은 힘든 시간을 보내고 있어요.<br/> 
UptoDoor 에서는 작은 개인가게부터 프랜차이즈까지, <br/>
우리동네의 구독서비스를 이용 함으로서 사장님에게 웃음을 선물해주세요. <br/>

# 🏡 Worth
### UptoDoor 는 추구합니다.
**❗️ No-stress**<br/>
매번 사기 아까운 물건들, 정기적으로 받아 볼 수 있어 스트레스가 줄어요.<br/>
**❗️ Comfortable**<br/>
내가 직접 하기 번거로운 일들, UptoDoor 구독 서비스가 챙겨주기 때문에 매우 편리해요.<br/>
**❗️ Win-win** <br/>
동네 사장님은 장기고객을 확보하며 매출상승 효과를, <br/>소비자들은 빠르고 편리한 구독서비스를 누리는 선 순환의 구조를 띄어요. <br/>
